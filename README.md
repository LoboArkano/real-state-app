## Getting Started

To get a local copy up and running follow these simple example steps.
```
    git clone git@github.com:LoboArkano/real-state-app.git
```

### Prerequisites

- npm

- Node.js

### Setup

Install dependencies with:

```
    npm install
```

Run the development server:

```
    npm run dev
```
